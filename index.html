Download APK v2

⚠️ Enable Install from unknown sources in your Android settings before installing.

How to Install

1
Download Ascend OS v2.apk to your Android device
2
Go to Settings → Apps → Special app access → Install unknown apps and enable it for your file manager
3
Open the downloaded APK file and tap Install
4
Sign in with your Google account to enable AI features & Firebase cloud sync
5
Go to Fitness → tap Grant Access to connect Health Connect for live health data
