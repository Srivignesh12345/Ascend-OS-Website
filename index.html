How to Install

1
Download the APK file to your Android device
2
Open Settings → Security → Unknown Sources and enable it
3
Open the downloaded APK file and tap Install
4
For Health data: install Health Connect from Play Store and grant permissions
5
Sign in with your Google account to enable AI features & cloud sync
